Get Elements Up To

Write a function called "getElementsUpTo".

Given an array and an index, "getElementsUpTo" returns a new array with all the elements after (but not including) the given index.

var output = getElementsUpTo(['a', 'b', 'c', 'd', 'e'], 2);

console.log(output); // --> ['a', 'b', 'c']
