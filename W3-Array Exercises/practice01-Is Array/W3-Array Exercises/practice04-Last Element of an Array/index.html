Last Element of an Array

Write a JavaScript function to get the last element of an array. Passing a parameter 'n' will return the first 'n' elements of the array.

input: last([7, 9, 0, -2]);

output: -2

input: last([7, 9, 0, -2],3);

output: [9, 0, -2]

input: last([7, 9, 0, -2],6);

output: [7, 9, 0, -2]
