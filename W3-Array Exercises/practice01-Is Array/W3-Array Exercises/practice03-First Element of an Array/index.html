First Element of an Array

Write a JavaScript function to get the first element of an array. Passing a parameter 'n' will return the first 'n' elements of the array.

input: first([7, 9, 0, -2]);

output: 7

input: first([], 3);

output: []

input: first([7, 9, 0, -2],3);

output: [7, 9, 0]

input: first([7, 9, 0, -2],6);

output: [7, 9, 0, -2]

input: first([7, 9, 0, -2],-3);

output: []
