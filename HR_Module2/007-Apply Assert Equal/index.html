Apply Assert Equal

Write an "assertEqual" function from scratch, from memory.

Examples

SUCCESS CASE

function square (n) { return n * n; }

assertEqual(square(4), 16, 'Square of 4');

// passed

FAILURE CASE

function square (n) { return (n * n); }

assertEqual(square(4), 1, 'Square of 4');

// FAILED [Square of 4] Expected "16", but got "1"
