Assert Equal

Write an assertEqual function from scratch.

It should compare actual and expected values with strict equality (not typecasting). Use console.log, not return statements.

Examples

SUCCESS CASE

function multiplyByTwo(n) { return n * 2; }

var output = multiplyByTwo(2); // returns 4

assertEqual(output, 4, 'it doubles 2 to 4');

// console output:

// passed

FAILURE CASE

function multiplyByTwo(n) { return (n * 2) + 1; // an incorrect implementation }

var output = multiplyByTwo(2); // returns 5

assertEqual(output, 4, 'it doubles 2 to 4');

// console output:

// FAILED [it doubles 2 to 4] Expected "4", but got "5"
